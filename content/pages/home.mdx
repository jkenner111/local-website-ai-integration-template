# Welcome

This is a placeholder home page. Replace it with content that describes
your community, organization, or business.

The content you're reading lives in `content/pages/home.mdx`. You can edit
it with any text editor, commit, and push — the site rebuilds automatically.

## What this template ships with

- A home page (this one), an about page, a contact form, events, and a
  calendar view.
- An AI-powered admin portal at `/admin` that lets non-technical users
  edit content by chatting in plain English.
- A push-to-deploy pipeline that rebuilds the site on your own server.

Edit `content/navigation.json` to change the menu, and drop additional
`.mdx` files into `content/pages/` to add pages.
